Explicación Etiquetas Formularios

    • Label

  • Listas desplegables
    • Select
    • Datalist
    • Option
    • Optgroup
  • Botones
  • Output

  • Fieldset: sirve para separar el formulario en secciones.

  • Legend: define un título para el formulario.

  • Label: engloba toda la parte referida a un campo del formulario, tanto el título como la caja donde escribiremos.

  • Input: crea controles interactivos para formularios con el fin de recibir datos del usuario.

  • Select: permite crear una lista desplegable, dentro de esta tendremos que poner diferentes "option".

  • Datalist: tiene el mismo funcionamiento que select, pero permite escribir y filtrar dentro de los elementos seleccionables.

  • Option: los option hacen referencia a cada uno de los valores dentro de un "select" o de un "datalist".

  • Optgroup: permite agrupar en diferentes secciones las opciones de un "select".

  • Textarea: es un cuadro de texto multilinea en el que el usuario podrá escribir.

  • Button: representa un elemento sobre el que podremos clicar.

  • Output: representa el resultado de un cálculo realizado.
